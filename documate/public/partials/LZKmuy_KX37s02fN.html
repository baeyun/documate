Documate

Converting your Markdown-docs into a fancy React-ish site.

Learn More Get Started

Markdown & HTML friendly

Documate converts your Markdown-based docs into a final HTML format that your end users can easily interact with. This saves you all the time to maintain docs. You can also inject custom HTML wherever necessary.

React-ish style

Your site is made to resemble React's official website. That emmersive feel when going through the docs is maintained while still being able to serve your docs.

Responsive

Across all viewports, Documate still maintains its UI style while providing a little bit more accessibility-related features for different viewports. Documate was built in a mobile-first sense with a powerful search feature.

Built with Create React App

Documate is built with the same tools you are familiar with. It currently operates on Create React App v2.1.2. You can easily incorporate it into your existing toolchain.
